# ChurnSense-AI

AI-Powered Telecom Customer Retention Intelligence System

## Overview
ChurnSense-AI is a business analytics and machine learning project focused on predicting customer churn in the telecom industry.

This project combines:
- Machine Learning
- SQL Analytics
- Data Visualization
- Business Intelligence
- Research Paper Reproduction

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- SQL
- Power BI
- Streamlit

## Goals
- Predict customer churn
- Analyze churn behavior
- Generate actionable business insights
- Reproduce research-driven methodologies
